Определение потока и процесса

Процесс - единица разделения системы, они 
как правило независимы
Поток - единица разделения внутри программы
В go есть горутины(облегченные потоки)
горутина - логическое объединение программы, которое выполняется 
независимо от остального

*go run -race <filename.go> -позволяет тестить горутины
(переключется в максимально хаотичнном порядке)
//ctrl + k + 0 сложить все

в блоке селект default выполняется если ни один канал не доступен для чтения 

явно завершить горутины можно через селект
но если горутин несколько то надо использовать sync

каналы надо создавать, когда есть данные к которым обращаются 
из нескольких мест
sync.Once позволяет выполнить один раз горутину(пр. инициализация)
